Тестовое задание (автотест):
1. Из кода тестов установить приложение
2. Открыть приложение
3. Добавить задачу
4. Перейти в задачу
5. Проверить, что заголовок и текст задачи соответствует введенным данным на шаге 3

Желателно использовать:
1. Maven
2. TestNG
3. Appium 
4. POM паттерн
5. PageFactory паттерн

Добавить возможность выбора типа устройства (эмулятор/девайс) через параметр maven
Разместить результат на GitHub

----------------------------------------------------------------

mvn test -DdeviceName=192.168.109.101:5555 -DappiumUrl=http://127.0.0.1:4723/wd/hub
